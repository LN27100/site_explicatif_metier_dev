Explication du codage

Ecrire du code permet la réalisation de site web statiques ou intéractifs. Cela se fait à l'aide d'un logiciel de code, tel que VScode, sur lequel on utilise différentes technologies et différents langages de développement. Ici, seuls les langages Html, Css et Javascript sont utilisés pour cet exemple de site rapide. J'ai également utilisé un frameworks, Bootstrap, qui est une bibliothèque en ligne de morceaux de codes réalisé par d'autres développeurs afin que l'on puisse les utiliser et gagner du temps dans la réalisation d'un site.
